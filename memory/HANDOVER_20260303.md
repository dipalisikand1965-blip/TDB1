## 🔄 MIRA AI INTEGRATION

### How Mira Knows Pet Info:
1. Soul Builder answers → `doggy_soul_answers` in pet profile
2. Pet profile → Mira context at chat time
3. Conversation → Stored in `conversation_memories`
4. Next chat → Memories retrieved and injected into prompt

### Rainbow Bridge Handling:
When `pet.rainbow_bridge = true`:
- Mira speaks in past tense
- Never suggests services/products
- Shows compassion: "I remember {pet_name}..."
- Soul profile marked as "preserved forever"
