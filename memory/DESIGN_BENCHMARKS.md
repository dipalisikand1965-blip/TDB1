# Mira Chat UI Design Benchmarks

## Reference Apps (Gold Standards)

### Apple iMessage / Messages
- **Gold standard for**: Clarity, spacing, animations, touch targets
- Clean bubble design with subtle shadows
- Generous padding and whitespace
- Smooth message send animations
- Clear read receipts and typing indicators
- Perfect touch target sizes (44pt minimum)

### WhatsApp
- **Benchmark for**: Simplicity and global familiarity
- Light background (#ECE5DD) with green accents
- Dual-check delivery indicators
- Simple, scannable message layout
- Time stamps grouped intelligently
- Minimal chrome, maximum content

### Slack
- **Benchmark for**: Desktop productivity chat (threads, reactions, structure)
- Clear sender attribution
- Threaded conversations
- Emoji reactions
- Rich formatting support
- Channel/context awareness in header

### ChatGPT
- **Benchmark for**: Long-form AI conversation readability
- Wide content area for AI responses
- Clear distinction between user/AI messages
- Code blocks with syntax highlighting
- Copy buttons on responses
- Markdown rendering for structure

---

## Design Principles for Mira

1. **Calm over chaotic** - Neutral backgrounds, no loud gradients
2. **Clarity over decoration** - Every element serves a purpose
3. **Breathing room** - Generous spacing, not cramped
4. **Hierarchy** - Clear visual distinction between message types
5. **Touch-friendly** - 44px minimum touch targets
6. **Readable** - High contrast, proper line heights
7. **Familiar** - Users should feel at home immediately
