## 🎯 ENHANCEMENT DETAILS

### E013: Remembered Service Providers
**What:** Mira remembers which groomers, vets, trainers the pet has used
**Why:** Reduces friction, builds trust, shows Mira truly "knows" the pet
**How:** 
- Store `service_history` in pet profile
- Query on relevant intents
- Offer as first option: "Should I book with [Provider Name] again?"
**Magic Moment:** "I see Buddy loved his last grooming at PawFect Spa. Should I book the same stylist?"

### E018: Birthday/Anniversary Reminders
**What:** Proactive notification about upcoming pet milestones
**Why:** Shows Mira cares about the pet's life, not just transactions
**How:**
- Store `birthday`, `gotcha_day` in pet profile
- Daily cron job checks upcoming dates (7 days, 3 days, 1 day)
- Push notification or in-app message
**Magic Moment:** "Buddy turns 5 in one week! 🎂 Would you like to start planning something special?"

### E021: Weather-Aware Suggestions
**What:** Adjust recommendations based on local weather
**Why:** Shows Mira is contextually aware, protects pet health
**How:**
- Get user location from profile
- Call weather API
- Adjust messaging: hot day → suggest cooling products, rainy → indoor activities
**Magic Moment:** "It's going to be 40°C today. I'd skip the afternoon walk and try some indoor puzzle games instead."

### E030: Memory Lane
**What:** Surface meaningful memories from the pet's journey
**Why:** Emotional connection, shows Mira remembers the relationship
**How:**
- Track significant events in `pet_timeline`
- On anniversaries, surface the memory
- Optional: parent can add photos to memories
**Magic Moment:** "Two years ago today, Buddy learned his first trick! Remember this video you shared?"
