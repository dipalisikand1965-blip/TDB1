🎁 MIRA'S BIRTHDAY BOX — MASTER SPECIFICATION

What Is It?

A curated 6-slot celebration box built specifically for one pet, based on their individual soul profile. No two boxes are the same. Mira selects every item based on her knowledge of the pet.

Where It Appears

Single dark card on /celebrate, positioned below the category strip and above the soul pillars.

Slot Intelligence Logic

SLOT 1 (Hero Cake):
  favourite food → [flavour] cake
  allergy → EXCLUDE, label "allergy-safe"
  breed fallback → Labrador=peanut butter, Indie=chicken, Shih Tzu=salmon

SLOT 2 (Joy Item by Pillar):
  Play → Favourite toy | Adventure → Outdoor kit | Social → Pawty kit
  Learning → Puzzle | Food → Treat platter | Grooming → Spa kit
  Health → Wellness pack | Memory → Photo props

SLOT 3 (Style):
  birthday registered → Custom embroidered bandana
  small dog → Birthday bow set
  large breed → Wide bandana

SLOT 4 (Memory):
  Memory score > 60 → Memory card + photo envelope
  exact birthday → Personalised date card
  multiple pets → Family card

SLOT 5 (Health) — MUST BE ALLERGY-SAFE:
  condition → Condition-specific supplement
  senior (>7) → Joint support
  puppy (<2) → Growth treat
  NO allergy data → Show notice in builder

SLOT 6 (Surprise by Archetype):
  Social Butterfly → Friend gift | Adventurer → Trail bandana
  Thinker → Hidden puzzle | Nurturer → Comfort plush
  Performer → Party hat + bow | Protector → Calming treat
  Free Spirit → Mystery toy (sealed)

Soul Discovery States

Soul %Behavior
< 30%Fallback items + "Tell Mira more" banner
30-70%Partial personalization
> 70%Full personalization, no caveats
89%+Full confidence, "exactly right"

Allergy Rules (CRITICAL)

  • Rule 1: Allergen in profile → EXCLUDE from ALL slots
  • Rule 2: Slot 1 (cake) MUST confirm "allergy-safe"
  • Rule 3: Slot 5 (health) MUST be allergen-checked
  • Rule 4: No allergy data → Show builder notice

Expansion Behaviour

  • Row-based inline expansion: Row 1 (pillars 1-4: Food, Play, Social, Adventure) expands below Row 1. Row 2 (pillars 5-8: Grooming, Learning, Health, Memory) expands below Row 2.
  • Only one open at a time — opening any pillar closes previously open one
  • Same pillar clicked again → toggles closed
  • Scroll: scrollIntoView({behavior:'smooth', block:'nearest'}) on open
  • Animation: height 0→auto, opacity 0→1, 300ms ease-in-out
