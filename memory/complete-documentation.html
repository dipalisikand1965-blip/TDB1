🐕 Platform Overview

MIRA OS: A hyper-personalized pet care platform with AI-powered recommendations, Soul Score profiling, and 14 lifestyle pillars.
Core Philosophy
  • Soul Score: Every pet has a unique profile built from 100+ data points
  • Breed Intelligence: AI recommendations based on breed-specific needs
  • Lifestyle Pillars: 14 pillars covering every aspect of pet care
  • Concierge: Human-in-the-loop support via WhatsApp/Email/Call
Tech Stack
Frontend:
  • React 18 with Vite
  • TailwindCSS + Shadcn/UI
  • React Router v6
  • Context API for state
Backend:
  • FastAPI (Python 3.11)
  • MongoDB Atlas (Motor async)
  • Cloudinary (image storage)
  • OpenAI GPT-4o (AI features)

  • celebration (106) = celebration packages/kits. NEVER use for Birthday Cakes.

Mira Imagines Card

For loved foods not in the cake collection. Dark gradient card with food emoji + "Request a Quote →" button.

  • Data source: pet.learned_facts filtered by type=loves/likes + category=preferences
  • Logic: check food word against cakes collection names → if not found → show imaginary card
  • Button click: fires openConcierge + openMiraAI events → concierge toast
  • Max 3 imaginary cards per session
Modal Footer — 2-State Live Narrative
  • Browsing: "✦ Everything here is personalised for {pet}" | "Explore More for {pet}"
  • Active (items added): "{emoji} + N things — {pet}'s plan is growing" | "Keep Building →"
  • addedCount increments on addToCart events while modal open; resets on category change
