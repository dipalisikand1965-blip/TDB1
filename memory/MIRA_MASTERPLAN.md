### 1. Richer Concierge Handoff Responses
**Problem**: Current response is just "I'll connect you with your pet concierge"
**Solution**: Even when handing off, Mira should:
- Acknowledge the concern empathetically
- Provide immediate safety tips if relevant
- Explain what the concierge will help with
- Show estimated response time

**Example - Current**:
> "I'll connect you with your pet concierge to help with this."

**Example - Should Be**:
> "Buddy's constant coughing needs attention. While I connect you with your concierge, here's what to watch for:
> - If coughing is accompanied by difficulty breathing → Emergency vet immediately
> - Kennel cough is common but treatable
> - Golden Retrievers can be prone to heart conditions that cause coughing
> 
> Your concierge will reach out within 1 hour to discuss next steps and recommend the right vet if needed.
> 
> [Chat with Concierge] [Find Emergency Vet]"

## 🟣 CONCIERGE EXCELLENCE (P1)

### Handoff Flow
```
User Query → Mira Analysis → 
  IF Complex/Emotional/Multi-step:
    → Create Structured Ticket
    → Send Admin Notification  
    → Send User Confirmation
    → Show ETA
    → Provide Interim Help
```

### Ticket Structure
```json
{
  "ticket_id": "CNC-20240206-0001",
  "priority": "high",
  "category": "health_concern",
  "pet": {
    "name": "Buddy",
    "breed": "Golden Retriever",
    "age": "5 years",
    "known_conditions": ["hip dysplasia"],
    "allergies": ["chicken"]
  },
  "request": "Dog constantly coughing",
  "mira_analysis": {
    "possible_causes": ["kennel cough", "allergies", "heart condition"],
    "urgency": "moderate - not emergency but needs attention",
    "recommended_action": "Vet consultation within 48 hours"
  },
  "suggested_response": "Recommend vet visit, ask about duration and other symptoms"
}
```

### Concierge Dashboard Needs
- All tickets in one view
- Pet context visible
- Mira's analysis visible
- Quick response templates
- WhatsApp integration
- Status tracking
