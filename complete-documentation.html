Product Overview

The Doggy Company's Pet Life Operating System is a pillar-based platform where each pillar (DINE, CARE, GO, CELEBRATE, etc.) is a fully personalised experience for a named dog — powered by Mira, the AI concierge who knows every dog by name, breed, size, health, and soul.

Core Philosophy: The soul profile is the dog's operating system. Every pillar reads from it and personalises everything — products, services, guided paths, and Mira's recommendations.

🌟 Soul Profile

94-question soul builder — breed, size, diet, anxiety, health, social preferences. Powers all pillar personalisation.

🤖 Mira Score Engine

Claude Sonnet scores products 0-100 for each pet. Auto-triggers on page load, cached in MongoDB.

🏡 Guided Paths

3-column card grid showing Mira-curated journeys. PathFlowModal with step-by-step guidance.

📦 Concierge Booking

All services booked via service desk tickets. "Hand to Concierge →" flow.

🤖 Mira Score Engine

The Mira Score Engine uses Claude Sonnet (via Emergent LLM Key) to score each product 0-100 for relevance to a specific dog's soul profile.

How It Works

  1. GoSoulPage mounts → auto-fires POST /api/mira/score-for-pet (fire-and-forget)
  2. Backend fetches pet soul profile + all go products
  3. Claude Sonnet scores each product: score (0-100), reasoning, mira_hint
  4. Scores stored in mira_product_scores MongoDB collection
  5. Next page load: GET /api/mira/claude-picks/{pet_id}?pillar=go returns sorted picks

First Visit Behavior

  • MiraPicksSection shows shimmer skeleton (4 ghost cards) for ~14 seconds while scoring runs
  • If no picks after timeout → shows "Mira Imagines" cards (personalized travel destination suggestions)
  • Second visit: real AI-scored picks appear
Go pillar was added to pillars_to_score in Session 66. Scoring now covers: dine, celebrate, care, go, fit, adopt.
