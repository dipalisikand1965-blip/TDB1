Via Admin Panel (No Code)

  1. 1. Go to /admin
  2. 2. Select pillar tab (e.g., care-cms)
  3. 3. Edit in the appropriate tab:
  4. • Page Settings - Title, Subtitle
  5. • Categories - Topic cards, images

  7. • Services - Related services
