☁️ CLOUDINARY SETUP GUIDE (MUST DO!)

⚠️ WHY THIS IS CRITICAL:
AI mockup images are stored as base64 (2MB each!) making Product Box VERY slow.
Cloudinary converts them to CDN URLs (100 bytes) = 10x faster loading!

📋 Step-by-Step Setup:

Step 1: Get Your API Keys from Cloudinary

  1. Go to console.cloudinary.com
  2. Click "View API Keys" button (bottom right of dashboard)
  3. Copy these 3 values:
    • Cloud Name: duoapcx1p (already known)
    • API Key: (numbers like 123456789012345)
    • API Secret: (click "eye" icon to reveal, copy the long string)

Step 2: Add to Backend Environment

Add these lines to /app/backend/.env:

CLOUDINARY_CLOUD_NAME=duoapcx1p
CLOUDINARY_API_KEY=your_api_key_here
CLOUDINARY_API_SECRET=your_api_secret_here

⚠️ Replace API_KEY and API_SECRET with YOUR actual values!

Step 3: Restart Backend

sudo supervisorctl restart backend

Step 4: Convert Existing Mockups

  1. Go to Admin Panel → Soul Products → Mockups tab
  2. Look for "Cloud Storage (Cloudinary)" section
  3. Click "Convert to Cloud (10)" button
  4. Repeat until "Base64: 0" (all converted!)

Rainbow Bridge Feature NEW

Handling pet loss with compassion and dignity.

How It Works

  1. Pet parent marks their pet as "crossed the rainbow bridge"
  2. Pet record is preserved (never deleted)
  3. UI shows gentle memorial indicators
  4. Mira AI becomes aware and responds with compassion
  5. No product suggestions - only emotional support

Mira's Response

When asked about a pet that has passed:

"I'm so sorry for your loss. Mystique crossed the rainbow bridge on March 7, 2026. She was such a calm, golden little Shih Tzu who loved liver treats and cheese... If you'd like, we can talk about her, write a tribute together, or think of a way to honor her memory."
