Pet Picker Flow (Post-Fix)

StepLogic
1. Message receivedGupshup POST → return 200 instantly → asyncio.create_task processes in background
2. DeduplicationCheck msg_id in wa_dedupe → skip if seen
3. Phone lookupQuery ALL emails linked to phone → fetch all pets
4. Single petSet wa_pet_state.active_pet → proceed to Mira AI
5. Multi-petCheck wa_pet_state.awaiting_pet_selection (fresh = <30 min)
6. Pet name in messageFuzzy match against pet list → set active pet → proceed
7. No matchFall through to get_mira_ai_response — clean re-picker, no "Sorry" dead end
