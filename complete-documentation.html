4. Mira Intelligence — Product Filtering

useMiraFilter.js — Single Source of Truth
/app/frontend/src/hooks/useMiraFilter.js
import { applyMiraFilter } from '../hooks/useMiraFilter';

// In any pillar page:
const filtered = applyMiraFilter(rawProducts, currentPet);
// Returns products with: mira_hint, _loved, _dimmed, miraPick, _healthSafe

// Render Mira's pick callout:
{filtered[0]?.miraPick && (
  <div style={{background:'linear-gradient(135deg,rgba(255,140,66,0.1),...', border:'1px solid rgba(255,140,66,0.3)', ...}}>
    <div>✦</div>
    <strong>Mira's pick:</strong> {filtered[0].name} — {filtered[0].mira_hint}
  </div>
)}
ALLERGEN_MAP (ingredient synonyms)
  • chicken → [chicken, poultry, fowl]
  • soy → [soy, soya, tofu, edamame]
  • wheat → [wheat, gluten, flour, barley]
  • dairy → [milk, cheese, butter, lactose, whey]
  • fish → [fish, salmon, tuna, cod, anchovy]
  • + eggs, beef, pork, lamb, shellfish
Ranking Priority
  • Rank 0 — Matches pet's favourite food/flavour
  • Rank 1 — Safe for health condition (treatment-safe)
  • Rank 2 — Explicitly allergen-free labelled
  • Rank 10 — General safe products
  • Rank 15 — Conflicts with nutrition goal (dimmed 55%)
  • Blocked — Contains known allergen (removed entirely)
⚠️ IMPORTANT: SharedProductCard automatically shows product.mira_hint as "Why Mira suggests this". Always run products through applyMiraFilter before passing to SharedProductCard.
