6. Key API Endpoints

Authentication

POST /api/auth/login     - User login (returns access_token)
POST /api/auth/register  - New user registration
GET  /api/auth/me        - Get current user profile

Service Flow

POST /api/service-requests           - Create service request (universal)
POST /api/concierge/mira-request     - Mira conversation ticket
POST /api/orders                     - Create order (auto-creates ticket)
GET  /api/concierge/requests         - Admin: Get all requests
GET  /api/concierge/stats            - Admin: Dashboard statistics
GET  /api/concierge/queue            - Admin: Service desk queue

Learn Content & Ask Mira

GET  /api/learn/guides       - Get learn guides
GET  /api/learn/videos       - Get learn videos
POST /api/os/learn/ask-mira  - Ask Mira AI a question (Learn page)

Products & Services

GET /api/products                    - Get products (supports category filter)
GET /api/service-box/services        - Get services by pillar
GET /api/membership/plans            - Get membership plans
POST /api/membership/create-order    - Create Razorpay payment order

Notifications

POST /api/whatsapp/test-notification - Test WhatsApp (welcome, booking, payment, membership)

🔄 MASTER SYNC Button

The MASTER SYNC button at the bottom performs 9 operations:

  1. Sync Shopify Products - Pull latest from store
  2. Product Intelligence - Add mira_hints & breed_tags
  3. AI Semantic Tagging - Smart product categorization
  4. Seed All Pillars - Products to all 14 pillars
  5. Seed Services Master - Sync services_master collection
  6. Seed Breed Services - Breed-specific services
  7. Seed Breed Tags - Breed metadata on products
  8. Add Mira Whispers - Personalization data
  9. Seed Production Data - FAQs, collections
