Admin CMS Guide

How to manage content on The Doggy Company website

Quick Start

1

Go to Admin Panel

Navigate to /admin and login with your credentials

2

Select a Pillar

Click on the pillar tab you want to edit (e.g., "Care CMS", "Dine CMS")

3

Edit Content

Use the tabs to modify different sections of the page

4

Save Changes

Click "Save All Changes" - changes appear on the website immediately

Understanding the CMS Tabs

Row 1: Basic Settings

Settings

Page title, subtitle, hero image. Use {petName} for personalization.

Ask Mira

The search bar at the top. Set placeholder text and button color.

Categories

Topic cards (like "Grooming", "Health"). Edit names, colors, images.

Products & Bundles

Select which products/bundles appear on this page.

Row 2: Advanced Settings

Services

Select which services appear in the services section.

Personalized

Toggle breed-smart, life-stage, and archetype recommendations.

Concierge

Premium concierge services with pricing and descriptions.

Mira Prompts

AI suggestions that appear proactively on the page.

Row 3: Content Sections (NEW!)

Help Buckets

"How can we help?" cards. 3 buckets with action items.

Daily Tips

Rotating tips. Add 7 for a full week rotation.

Guided Paths

Step-by-step journeys. Each path has 5 steps.

How to Edit Help Buckets

Help Buckets are the "How can we help?" cards on each page. Each bucket has:

  • Title: The bucket heading (e.g., "Grooming Help")
  • Icon: Choose from Sparkles, Heart, PawPrint, Award, etc.
  • Color: Pink, Teal, Amber, Blue, Purple, Green
  • Items: Clickable action items that open Mira chat

Tip: Most pages have 3 help buckets with 3-4 items each.

How to Edit Daily Tips

Daily Tips rotate based on the day. Users see a different tip each day.

  • Category: Grooming, Health, Training, Behavior, etc.
  • Tip Content: The actual tip text (keep it actionable and useful)
  • Color: Gradient color for the tip card

Tip: Add 7 tips for a full week rotation. The tip shown depends on the current day.

How to Edit Guided Paths

Guided Paths are step-by-step learning journeys for users.

  • Title: Path name (e.g., "New Pet Parent Path")
  • Topic Slug: Which topic the path relates to (for navigation)
  • Color: Pink, Amber, Purple, Blue, Green, Red, Teal
  • Steps: The journey milestones (add 5 steps per path)

Tip: Most pages have 6 guided paths with 5 steps each.

Using Personalization Variables

Use these placeholders in titles and text - they'll be replaced with actual values:

Variable What it becomes
{petName}The user's pet name (e.g., "Mojo")
{breedName}The pet's breed (e.g., "Shih Tzu")

Example: "Caring for {petName} made simple" becomes "Caring for Mojo made simple"

Best Practices

  • DO: Save frequently while editing to avoid losing work
  • DO: Test your changes on the live page after saving
  • DO: Keep tips actionable and specific
  • DON'T: Leave fields empty - use defaults if unsure
  • DON'T: Use very long text in tip cards (keep under 200 characters)
